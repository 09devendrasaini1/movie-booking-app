# 🎬 Movie Booking App

A full-stack MERN application for booking movie tickets online.

## 🚀 Live Demo
https://movie-booking-apk.netlify.app

## 💻 GitHub Repository
https://github.com/09devendrasaini1/movie-booking-app

## Features
- Select a movie
- Choose a show time
- Book seats
- View last booking details
- Responsive UI

## Tech Stack
- React.js
- Node.js
- Express.js
- MongoDB Atlas
- Netlify
- Render
